# NodeJS ToDo List App

## Descripción

Esta aplicación permite gestionar tareas en una lista. Está desarrollada con NodeJS y Express, y utiliza una base de datos MongoDB.
## Características
- Añadir nuevas tareas
- Editar tareas existentes
- Eliminar tareas
